# Weather Dashboard

A simple React + Node.js app to fetch and display current weather information using a city name.


## Tech Stack

- **Frontend:** React.js
- **Backend:** Node.js, Express
- **API Source:** [OpenWeatherMap](https://openweathermap.org/)
